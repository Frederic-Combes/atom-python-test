## Description

* Run py.tests tests on Atom.

## Requirements

* You need py.test installed to use this package: 

    ```
    pip install pytest
    ```
    
## Usage

1) Running all tests ```(Ctrl + Alt + T)```

2) Run test under cursor ```(Ctrl + Alt + C)```
